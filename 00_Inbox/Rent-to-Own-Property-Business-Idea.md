# ไอเดีย: ธุรกิจให้เช่าบ้านที่พร้อมขาย / Rent-to-Own

## รูปแบบธุรกิจที่เป็นไปได้

### 1. Rental with Option to Buy
- เช่าระยะยาว (1-3 ปี) พร้อมสิทธิ์ซื้อก่อนหมดสัญญา
- เงินเช่าบางส่วนนับเป็นเงินดาวน์

### 2. Leasehold to Freehold
- ซื้อทรัพย์ที่มี leasehold แล้วต่อเป็น freehold

### 3. Pre-construction Rental
- เช่าคอนโด/บ้านที่กำลังสร้าง ระหว่างรอโอนกรรมสิทธิ์

### 4. Seller Financing
- ผู้ขายเป็นคนให้กู้เอง (ผ่อนต่อกับเจ้าของเดิม)

## สิ่งที่ต้องศึกษา

- [ ] กฎหมายที่เกี่ยวข้อง (สัญญาเช่าซื้อ, การโอนกรรมสิทธิ์)
- [ ] ต้นทุนและ ROI ที่คาดหวัง
- [ ] ความเสี่ยงด้านกฎหมายและการเงิน
- [ ] กลุ่มเป้าหมาย (ผู้เช่าที่อยากเป็นเจ้าของ)
