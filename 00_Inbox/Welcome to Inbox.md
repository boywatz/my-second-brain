# Welcome to Inbox 📥

นี่คือจุดเริ่มต้นของทุกไอเดีย!

## วิธีใช้ Inbox
- **จดทุกอย่างที่นึกออก** ลงที่นี่ก่อน — ไม่ต้องจัดหมวดหมู่
- **ตรวจสอบเป็นประจำ** (แนะนำทุกวัน หรืออย่างน้อยทุกสัปดาห์)
- **ย้ายไปที่ที่เหมาะสม** ตามหลัก PARA:
  - กำลังทำอยู่ → `01_Projects`
  - ดูแลต่อเนื่อง → `02_Areas`
  - น่าสนใจแต่ยังไม่ใช้ → `03_Resources`
  - จบแล้ว → `04_Archive`

## Tips
- ใช้ Ctrl/Cmd + N สร้างโน้ตใหม่ได้เลย
- พิมพ์ `[[` เพื่อลิงก์ไปหาโน้ตอื่น
- ใช้ tag `#to-process` สำหรับโน้ตที่ยังไม่ได้สรุป
