# Caveman Plugin

Caveman is a plugin that helps reduce LLM token usage.

## Overview

- **Purpose**: Minimize token consumption when interacting with LLMs
- **Mechanism**: Compresses/shrinks prompts while preserving essential information

## Use Cases

- Cost optimization for API calls
- Faster response times (less data to process)
- Working within token limits for context-heavy tasks

## Notes

- Check documentation for specific compression algorithms
- May have trade-offs in response quality depending on compression level
